                                                     kmeans聚类
1.问题描述

应用K-means算法实现三维空间点的聚类。数据放在数据文件中，第一行为数据的个数，以后各行是各个点的x,y,z坐标。

2. 实验要求

编写程序（语言不限），读取文本文件中的数据，并用K-means算法输出每个聚类中心及各个聚类（簇）中每个点的坐标。
要求程序符合结构化程序设计风格，关键地方需加注释。
